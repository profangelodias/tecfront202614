Criar uma página: - Criar um arquivo "mista.html" - Adicionar o conteúdo HTML básico (DOCTYPE, html, head, body) - Adicionar um header com título e nav - Adicionar um main com um formulário livre (pode ser qualquer coisa, não precisa ser igual ao forms.html) - Adicionar um footer com um texto de copyright Obs.: tem que seguir a mesma estrutura das outras páginas (index.html, forms.html, resultados.html) para manter a consistência do site. Obs.: informar para a IA não colocar class nem id. Atividade: - Colocar ids em campos específicos e alterar o estilo desses campos; - Colocar classes em campos específicos e alterar o estilo desses campos; E GUARDEM ESSE CÓDIGO PARA O PRÓXIMA AULA, VAMOS USAR ELE LÁ!
